# Uso diario

Esta sección describe el uso habitual de Mmeter en el día a día.
Aquí se explica el funcionamiento normal del equipo y el sistema de notificaciones.

---

## Funcionamiento normal

En condiciones normales, Mmeter se utiliza en **modo medición**.

Al encender el equipo:
- Mmeter inicia automáticamente la medición.
- La app muestra los valores en tiempo real.
- No es necesario realizar acciones adicionales para comenzar a medir.

Durante el uso diario:
- El equipo permanece midiendo de forma continua.
- Los valores se actualizan automáticamente.
- El usuario puede supervisar las magnitudes desde la app en cualquier momento.

Este es el modo de funcionamiento habitual del sistema.

## Notificaciones push

</a>

Mmeter utiliza **notificaciones push** para informar eventos relacionados con las **zonas de cada magnitud medida**.

Las notificaciones se generan cuando un valor entra en una de las siguientes zonas:
- **LOW** (valor bajo)
- **OK** (valor dentro del rango normal)
- **HIGH** (valor alto)

Por cada cambio de zona, el sistema genera un evento que:
- Envía una **notificación push a la app**.
- Envía un **correo electrónico** asociado a la cuenta.

Las notificaciones permiten al usuario estar informado del estado de las mediciones sin necesidad de abrir la app constantemente.

> Las alertas son gestionadas por la plataforma y enviadas automáticamente por el sistema.

### Evento de encendido

Existe un evento de **encendido del equipo (power-up)**.
Este evento es **configurable** y **no se encuentra activo por defecto**.

La configuración de este evento se explica en la sección de **[Configuración avanzada](/docs/config-avanzada)**.
