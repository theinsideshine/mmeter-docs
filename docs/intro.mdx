# Introducción

Mmeter es un medidor ambiental conectado, diseñado para monitorear de forma continua las condiciones del entorno y mostrar la información de manera clara en la app.

Funciona de forma automática y está pensado para operar siempre encendido, sin intervención diaria.

---

## Características de Mmeter

Estas son las funciones principales de Mmeter. (Más adelante vamos a ajustar esta lista cuando cerremos el alcance final del producto).

- Medición de **temperatura**
- Medición de **humedad**
- Medición de **luz**
- Cálculo automático de **VPD (Déficit de Presión de Vapor)**
- **Alertas y notificaciones** cuando un valor sale de los rangos configurados
- Visualización desde la **app**
- Conectividad **Wi-Fi** para sincronización de datos y alertas
- Recuperación automática ante cortes de conexión (el equipo reinicia y espera que la red vuelva)

## Requisitos de Internet

Para funcionar correctamente, Mmeter necesita:

- Red Wi-Fi de **2.4 GHz**
- Conexión a Internet doméstica o similar
- Redes sin portales cautivos ni autenticación mediante navegador
